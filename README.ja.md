## 🔍 プロジェクトについて

**OSINT Brasil** は、情報専門家、調査員、ジャーナリスト、サイバーセキュリティ研究者向けの完全なオープンソースプラットフォームです。**1,300以上のオープンソースインテリジェンス (OSINT) ツール**を **53カテゴリ**に整理し、**30言語**に対応。**Webアプリ、Windows デスクトップアプリ、Android アプリ**として利用可能です。

ツールデータは GitHub の [awesome-osint](https://github.com/jivoi/awesome-osint) リポジトリから自動更新されます。

---

## ✨ 機能

- **🔄 自動更新** — 起動時に GitHub から同期、オフラインキャッシュにフォールバック
- **🌍 30言語対応** — ポルトガル語、英語、スペイン語、フランス語、ドイツ語、ロシア語、中国語、日本語、アラビア語など
- **📱 クロスプラットフォーム** — Web、Windows、Android
- **🔍 リアルタイム検索** — 1,300以上のツールを名前・説明・タグで検索
- **📂 53カテゴリ** — テーマ別グループ、展開可能なサイドバー
- **⭐ お気に入り** — ローカルストレージで永続化
- **🎨 5種の表示モード** — グリッド、リスト、テーブル、ツリー、コンパクト
- **🌙 ダーク/ライトテーマ** — localStorage に保存
- **♿ アクセシビリティ** — ARIA ラベル、スキップリンク、完全 RTL サポート

## 🚀 クイックスタート

```bash
git clone https://github.com/azurejoga/osint-explorer.git
cd osint-explorer
npm install
npm run dev
```

`http://localhost:5173` を開く

## 💾 インストール

### 🌐 Web版
[osint.juanmathewsrebellosantos.com](https://osint.juanmathewsrebellosantos.com) にアクセス — インストール不要。

### 💻 デスクトップ版（Windows）

[リリースページ](https://github.com/azurejoga/osint-explorer/releases) からダウンロード：

| ファイル | 種別 |
|---|---|
| `OSINT Tools Explorer Setup X.X.X.exe` | インストーラー |
| `OSINT Tools Explorer X.X.X.exe` | ポータブル版（インストール不要） |

### 📱 Android

[リリースページ](https://github.com/azurejoga/osint-explorer/releases) から APK をダウンロードし、Android デバイスに直接インストール（設定→セキュリティで「提供元不明のアプリ」を有効化）。

## 🔧 ビルド

```bash
# フルビルド
npm run build

# Windows のみ
npm run build:electron

# Android のみ（Android Studio 必須）
npm run build:android

# Web のみ
npm run build:web
```
