## 🔍 Acerca del Proyecto

**OSINT Brasil** es una plataforma open-source para profesionales de inteligencia, investigadores, periodistas e investigadores de ciberseguridad. Reúne **más de 1.300 herramientas de Open Source Intelligence (OSINT)** en **53 categorías**, con soporte para **30 idiomas**, disponible como **app web, desktop Windows y app Android**.

Las herramientas se actualizan automáticamente desde el repositorio [awesome-osint](https://github.com/jivoi/awesome-osint) en GitHub.

---

## ✨ Funcionalidades

- **🔄 Actualización automática** — Sincroniza con GitHub al abrir, fallback offline
- **🌍 30 idiomas** — Portugués, Inglés, Español, Francés, Alemán, Ruso, Chino, Japonés, Árabe y más
- **📱 Multiplataforma** — Web, Windows y Android
- **🔍 Búsqueda en tiempo real** — Por nombre, descripción y etiquetas en 1.300+ herramientas
- **📂 53 categorías** — Grupos temáticos con barra lateral expandible
- **⭐ Favoritos** — Persistencia local (localStorage)
- **🎨 5 modos de visualización** — Cuadrícula, Lista, Tabla, Árbol, Compacto
- **🌙 Tema oscuro/claro** — Persistido en localStorage
- **♿ Accesibilidad** — Etiquetas ARIA, skip links, soporte RTL completo

## 🚀 Inicio Rápido

```bash
git clone https://github.com/azurejoga/osint-explorer.git
cd osint-explorer
npm install
npm run dev
```

Abrir `http://localhost:5173`

---

## 💾 Instalación

### 🌐 Web
Accede en [osint.juanmathewsrebellosantos.com](https://osint.juanmathewsrebellosantos.com) — sin instalación.

### 💻 Desktop (Windows)

Descarga en la [página de releases](https://github.com/azurejoga/osint-explorer/releases):

| Archivo | Tipo |
|---|---|
| `OSINT Tools Explorer Setup X.X.X.exe` | Instalador |
| `OSINT Tools Explorer X.X.X.exe` | Portátil (sin instalación) |

### 📱 Android

Descarga el APK en la [página de releases](https://github.com/azurejoga/osint-explorer/releases) e instálalo directamente (habilita "Fuentes desconocidas" en Ajustes).

---

## 🔧 Build

```bash
# Build completo
npm run build

# Solo Windows
npm run build:electron

# Solo Android (requiere Android Studio)
npm run build:android

# Solo Web
npm run build:web
```
