## 🔍 项目简介

**OSINT Brasil** 是一个完整的开源平台，专为情报专业人员、调查员、记者和网络安全研究人员设计。收录了 **1,300+ 开源情报 (OSINT) 工具**，分为 **53 个类别**，支持 **30 种语言**，可用作 **Web应用、Windows桌面应用和Android应用**。

工具数据自动从 GitHub 上的 [awesome-osint](https://github.com/jivoi/awesome-osint) 仓库更新，确保目录始终保持最新。

---

## ✨ 功能特点

- **🔄 自动更新** — 每次启动时从 GitHub 同步，支持离线缓存备用
- **🌍 30种语言** — 葡萄牙语、英语、西班牙语、法语、德语、俄语、中文、日语、阿拉伯语等
- **📱 跨平台** — Web、Windows 和 Android
- **🔍 实时搜索** — 在1,300+工具中按名称、描述和标签搜索
- **📂 53个分类** — 按主题分组，可展开的侧边栏
- **⭐ 收藏夹** — 本地持久化存储
- **🎨 5种视图模式** — 网格、列表、表格、树形、紧凑
- **🌙 深色/浅色主题** — 保存在 localStorage
- **♿ 无障碍支持** — ARIA 标签、跳转链接、完整 RTL 支持

## 🚀 快速开始

```bash
git clone https://github.com/azurejoga/osint-explorer.git
cd osint-explorer
npm install
npm run dev
```

打开 `http://localhost:5173`

---

## 💾 安装

### 🌐 网页版
访问 [osint.juanmathewsrebellosantos.com](https://osint.juanmathewsrebellosantos.com) — 无需安装。

### 💻 桌面版（Windows）

在 [releases页面](https://github.com/azurejoga/osint-explorer/releases) 下载：

| 文件 | 类型 |
|---|---|
| `OSINT Tools Explorer Setup X.X.X.exe` | 安装程序 |
| `OSINT Tools Explorer X.X.X.exe` | 便携版（免安装） |

### 📱 Android

在 [releases页面](https://github.com/azurejoga/osint-explorer/releases) 下载APK，直接安装到Android设备（需在设置中启用"未知来源"）。

---

## 🔧 构建

```bash
# 完整构建
npm run build

# 仅 Windows
npm run build:electron

# 仅 Android（需要 Android Studio）
npm run build:android

# 仅 Web
npm run build:web
```
